## Platform Overview

Learning Adventures is an educational gaming platform designed for K-5 students. The platform offers:

- **41+ Educational Games & Lessons** across Math, Science, English, History, and Interdisciplinary subjects
- **COPPA-Compliant Child Accounts** with parental controls
- **Progress Tracking** with achievements and badges
- **Multi-Device Support** for learning anywhere
